# Library - A Personal Book Management app
> A library app where you can manage books by add, delete and change read status. Built with HTML, CSS &amp; Javascript

![Home page](https://i.imgur.com/mtty45d.png)

## Features

- Allows user to add new book to library

- User can change their read status of the book.

- User can delete the book from their library

## Built With

- HTML
- CSS
- Javascript
